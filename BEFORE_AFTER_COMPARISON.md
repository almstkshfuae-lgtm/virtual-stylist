## Code Quality Improvements

### Layout Structure

**BEFORE:**
```tsx
<div className="flex items-stretch">
  <div className="md:w-5/12">Empty grey panel</div>
  <div className="md:w-7/12">Content (awkward proportions)</div>
</div>
```

**AFTER:**
```tsx
<div className="flex flex-col items-center justify-center">
  <div className="max-w-2xl">Centered, responsive content</div>
</div>
```

### Responsive Classes

**BEFORE:**
- Limited responsive handling
- Not optimized for mobile
- Breakpoints not strategic

**AFTER:**
- `sm:` for phones landscape
- `md:` for tablets
- `lg:` for desktops
- Proper cascading of styles

### Components

**BEFORE:**
```tsx
const FeatureCard = () => (
  <div className="p-6 rounded-2xl">
    {/* Static sizing */}
  </div>
);
```

**AFTER:**
```tsx
const FeatureCard = () => (
  <div className="p-6 sm:p-8 rounded-2xl sm:rounded-3xl">
    {/* Responsive sizing across breakpoints */}
  </div>
);
```
