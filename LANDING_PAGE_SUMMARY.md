## How to Test

### Quick Test (5 minutes)
```bash
npm run dev
```
Then:
1. Open http://localhost:5173
2. Resize browser from 320px to 1920px
3. Toggle dark/light mode
4. Try language selector
5. Test all buttons

### Comprehensive Test (30 minutes)
1. Test on actual devices (phone, tablet, desktop)
2. Test landscape orientation
3. Test with keyboard navigation (Tab through elements)
4. Test with screen reader
5. Check page in both light and dark modes
